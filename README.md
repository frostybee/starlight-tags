A plugin to add tags to your Starlight docs.

## Package

If you are looking for the Starlight plugin package, you can find it in the [`packages/starlight-tagging/`](/packages/starlight-tagging/) directory.

## Project structure

This project uses pnpm workspaces to develop a single Starlight plugin from the `packages/starlight-tagging/` directory. A Starlight documentation site is also available in the `docs/` directory that is also used for testing and demonstrating the Starlight plugin.
