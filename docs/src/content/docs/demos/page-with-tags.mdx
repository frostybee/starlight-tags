import { PageWithTags } from 'starlight-tags/components';

This page demonstrates the `PageWithTags` component, which wraps content and displays tags at configurable positions.

:::tip[Wrapper Component]
This component wraps your content and automatically fetches tags from the page's frontmatter, or you can specify custom tags via `tagIds`.
:::

## Bottom Position (Default)

Tags appear after the content with a separator line and "Tags" heading:

<PageWithTags position="bottom" tagIds={['webhooks', 'intermediate']}>
  <p>This is sample content wrapped by PageWithTags.</p>
</PageWithTags>

```astro
import { PageWithTags } from 'starlight-tags/components';

<PageWithTags position="bottom">
  <YourContent />
</PageWithTags>
```

## Top Position

Tags appear before the content:

<PageWithTags position="top" tagIds={['webhooks', 'intermediate']}>
  <p>This is sample content wrapped by PageWithTags.</p>
</PageWithTags>

```astro
<PageWithTags position="top">
  <YourContent />
</PageWithTags>
```

## Both Positions

Tags appear at both top and bottom:

<PageWithTags position="both" tagIds={['webhooks', 'intermediate']}>
  <p>This is sample content wrapped by PageWithTags.</p>
</PageWithTags>

```astro
<PageWithTags position="both">
  <YourContent />
</PageWithTags>
```

## Without Title

Hide the "Tags" heading at the bottom:

<PageWithTags position="bottom" showTitle={false} tagIds={['webhooks', 'intermediate']}>
  <p>This is sample content wrapped by PageWithTags.</p>
</PageWithTags>

```astro
<PageWithTags position="bottom" showTitle={false}>
  <YourContent />
</PageWithTags>
```

## Custom Tag Selection

Specify which tags to display using `tagIds`:

<PageWithTags position="bottom" tagIds={['stable', 'sdk-python', 'advanced']}>
  <p>This content shows custom-selected tags.</p>
</PageWithTags>

```astro
<PageWithTags position="bottom" tagIds={['stable', 'sdk-python', 'advanced']}>
  <YourContent />
</PageWithTags>
```
