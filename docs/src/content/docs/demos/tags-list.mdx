This page demonstrates the `TagsList` component, which renders a collection of tags with customizable layouts and sorting.

## Basic Usage

Display all tags in the system:

<TagsListDemo limit={8} />

## Layouts

### Inline (Default)

Tags flow horizontally and wrap:

<TagsListDemo layout="inline" limit={6} />

```astro
<TagsList tags={tags} layout="inline" />
```

### Grid

Tags in a responsive grid:

<TagsListDemo layout="grid" limit={6} />

```astro
<TagsList tags={tags} layout="grid" />
```

### List

Tags stacked vertically:

<TagsListDemo layout="list" limit={6} />

```astro
<TagsList tags={tags} layout="list" />
```

## With Title

Add a heading above the tags:

<TagsListDemo title="Related Topics" limit={5} />

```astro
<TagsList tags={tags} title="Related Topics" />
```

### Solid (Default)

## With Page Counts

Show how many pages use each tag:

<TagsListDemo showCount={true} limit={6} />

```astro

```

## Specific Tags

Display only specific tags by ID:

<TagsListDemo tagIds={['authentication', 'rest-api', 'webhooks', 'stable']} />

```astro
---
const specificTags = ['authentication', 'rest-api', 'webhooks', 'stable']
  .map(id => getTag(id))
  .filter(Boolean);
---

<TagsList tags={specificTags} />
```
