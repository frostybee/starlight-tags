This page demonstrates the `TagBadge` component, which renders a single tag as a styled badge.

## Basic Usage

Single tags with different configurations:

<div style="display: flex; gap: 0.5rem; flex-wrap: wrap; align-items: center;">
  <TagBadge tagId="authentication" />
  <TagBadge tagId="rest-api" />
  <TagBadge tagId="webhooks" />
  <TagBadge tagId="stable" />
</div>

```astro
import { TagBadge } from 'starlight-tags/components';

<TagBadge tagId="authentication" />
```

## Style Variants

### Solid (Default)

Filled background with the tag's color:

<div style="display: flex; gap: 0.5rem; flex-wrap: wrap; align-items: center;">
  <TagBadge tagId="authentication" variant="solid" />
  <TagBadge tagId="rest-api" variant="solid" />
  <TagBadge tagId="beginner" variant="solid" />
</div>

### Outline

Bordered with transparent background:

<div style="display: flex; gap: 0.5rem; flex-wrap: wrap; align-items: center;">
  <TagBadge tagId="authentication" variant="outline" />
  <TagBadge tagId="rest-api" variant="outline" />
  <TagBadge tagId="beginner" variant="outline" />
</div>

### Ghost

Subtle background tint:

<div style="display: flex; gap: 0.5rem; flex-wrap: wrap; align-items: center;">
